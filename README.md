# Portal de clases online: CodeForTheSpace

## Introducción

Proyecto final del Módulo 1 del Máster de Desarrollo Web ATR.

Este proyecto simula el proceso de desarrollo dentro de una empresa
ficticia llamada CodeForTheSpace, en el que se pone en práctica todo lo 
aprendido sobre Git.



## Estructura de ficheros

CodeForTheSpace/
├── README.md
├── courses/
│   └── courses.md
├── landing/
│   └── landing.md
├── logic/
│   ├── logic.css
│   ├── logic.html
│   └── logic.md
└── payment/
    ├── confirmation.md
    ├── payment.md
    └── receipt.md



## Tecnologías utilizadas

- Git
- WSL - Ubuntu
- VisualStudioCode
- CSS
- HTML
- MarkDown



## Herramientas necesarias

Para poder visualizar el contenido html y css, es recomendable instalar en VSCode la extensión Live Server.



## Instalación 

Tras su descarga, descomprimir carpeta del proyecto. Y abrirla con VSCode.
